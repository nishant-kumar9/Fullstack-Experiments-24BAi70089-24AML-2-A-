# Full Stack Authentication System (JWT + RBAC)

## 📌 Description
This project implements a full-stack authentication system using React, Node.js, Express, and MongoDB Atlas. It includes JWT authentication and role-based access control (RBAC).

---

## 🚀 Features
- Login form with validation (React + Material UI)
- JWT-based authentication
- Protected routes
- Role-based access (Admin/User)
- MongoDB Atlas database
- Logout functionality

---

## 🛠️ Tech Stack
- Frontend: React.js, Material UI
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: JSON Web Token (JWT)
